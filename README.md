
## Jak uruchomić lokalnie

1. Pobierz repozytorium:
   ```bash
   git clone https://github.com/WiktorBanasyk/trigonometric-functions.git
   ```
2. Wejdź do folderu:
   ```bash
   cd trigonometric-functions
   ```
3. Otwórz `index.html` w przeglądarce.
